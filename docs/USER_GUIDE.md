# User Guide

Welcome to the Farnsworth Morse Code Learner. This app is designed to get you proficient in Morse Code by teaching your brain to recognize character rhythms rather than counting dots and dashes.

## The Koch Method
The app uses the Koch Method for introducing new characters.
1. You start with only two characters (e.g., K and M).
2. The app plays a character, and you type what you heard.
3. Once you achieve 90% accuracy with the current set of characters, a new character is introduced.
4. This ensures you master characters at full speed before adding complexity.

## The Farnsworth Method
In the Settings menu, you will see two speed dials:
- **Character Speed (Wc)**: How fast the individual "beeps" play. **Keep this high (e.g., 20 WPM)**.
- **Effective Speed (We)**: How much silence is placed between characters. **Start this low (e.g., 10 WPM)**.

By keeping the Character Speed high, you learn the true "sound" of the character. As you get better, slowly increase the Effective Speed until it matches the Character Speed.

## Playing the Game
1. Tap **Start Drill**.
2. Listen to the audio tone (and feel the haptic pulse).
3. Type the character using the on-screen keyboard.
4. **Pro Tip**: You don't have to wait for the audio to finish! If you recognize the character immediately, type it. The app will buffer your input and advance automatically.

## Accessibility
The app is fully compatible with VoiceOver. Status updates, correct/incorrect feedback, and keyboard input are seamlessly announced. Make sure your device isn't on silent mode, though the app will override the silent switch for Morse audio.
